Какой минимум нужен казначею?

Выбери экраны для MVP — всё остальное потом

1

Вход (логин)

Страница авторизации. Обязательна.

2

Главная (дашборд)

Сводка: общий долг по организации, последние платежи, счётчики. Быстрый обзор.

3

Участки + долги

Таблица: номер участка, владелец, телефон, долг. Основной рабочий список казначея.

4

Принять платёж

Форма: выбрать члена, сумма, дата, ЕРИП-референс. Провести кнопкой.

5

Члены кооператива

Список физлиц: ФИО, телефон, активность, баланс лицевого счёта.

6

Должники

Только те, у кого долг > 0. Сортировка по сумме. Для работы с задолженностями.

7

Журнал документов

История всех операций: платежи, начисления, сторно. Фильтр по типу и дате.

Совет

Для старта достаточно: Вход → Участки+долги → Принять платёж. Остальное добавим потом. Выбери что важно для первого запуска.
